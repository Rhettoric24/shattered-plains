# Convex Scaffold

This folder is the planned online multiplayer backend for the Shattered Plains prototype.

It is a scaffold right now. The local playable game still runs from `outputs/`.

Once Convex is installed and initialized, this folder will hold the live backend functions and database schema.

Useful future commands:

```bash
npm install convex
npx convex dev
```

Those commands require internet access and a Convex account, so they are intentionally not run automatically here.
