# Shattered Plains Prototype

A browser-playable multiplayer strategy prototype using Convex for accounts, database state, and scheduled game events.

## Current Convex Version

Start the local static server:

```bash
npm start
```

Then open:

```text
http://127.0.0.1:4180/
```

Build the deployable static site:

```bash
npm run build
```

Preview the deployable build:

```bash
npm run preview
```

## Main Folders

- `outputs/` contains the current browser app source files.
- `dist/` is generated by `npm run build` and contains static deploy files.
- `convex/` contains the online multiplayer backend.
- `scripts/` contains build tooling for the static frontend.
- `.github/workflows/` contains the GitHub Pages deployment workflow.
- `docs/` contains planning notes.

## Live Gameplay Notes

- GitHub Pages serves the browser app.
- Convex owns accounts, player state, game actions, raids, Plateau Runs, messages, and admin-only testing tools.
- Plateau Runs are checked every five minutes and open during the noon and 8 PM Mountain time windows.
- The dashboard includes a World Alerts strip for open Plateau Runs, incoming raids, unread messages, and outgoing raids.
